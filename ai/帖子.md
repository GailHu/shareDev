# World Monitor
```markdown
World Monitor是一个统一的情报态势感知平台，由独立开发者koala73（真实作者Elie Habib）打造。它把海量公开数据（OSINT）整合成一个美观、高效、交互式的3D仪表盘，支持三种变体：• World版：专注地缘政治、冲突、军事• Tech版：AI、数据中心、半导体供应链• Finance版：市场、加密货币、稳定币、预测市场核心亮点：• 3D WebGL地球仪（deck.gl驱动，丝滑60fps）• AI自动合成情报（支持Ollama本地大模型，完全离线隐私）• PWA一键安装 + 桌面原生App（Tauri）• 16种语言全界面本地化（含简中、繁中，RTL支持）• 完全离线可用（地图缓存 + 本地LLM）它不是玩具，而是一个生产级工具：政府级工具现在开源免费了！
项目地址：https://github.com/koala73/worldmonitor

```
