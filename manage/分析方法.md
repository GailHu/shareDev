# 一、四色分析法
> 优势：分类清晰、逻辑简单，明确角色、实体、资料、属性，便于业务梳理、责任划分与台账管理。
> 局限：分析维度单一，缺乏深度；主观性强，标准不统一；不适用于创新业务与复杂系统整体分析。
# 二、问题根源类
- 5Why 分析法：连续逐层追问，深挖问题根本原因，适合事故、隐患复盘。
- 鱼骨图（因果分析）：从人、机、料、法、环、测多维度，全面梳理问题影响因素。
- 冰山分析法：区分表面现象与深层隐性问题，侧重管理、制度根源剖析。
- 二八分析法：聚焦关键少数主要矛盾，优先解决核心问题，提升管控效率。
# 三、战略环境类
- SWOT：分析优势、劣势、机会、威胁，用于项目研判、整体发展研判。
- PEST：从政治、经济、社会、技术，分析宏观外部环境与行业形势。
- 波特五力：研判行业竞争格局、上下游及替代风险，适用于行业分析。
# 四、流程业务类
- 5W1H：围绕何事、何故、何人、何时、何地、如何，全面完整梳理事件与流程。
- 泳道图分析：按部门、角色划分边界，厘清跨部门流程与岗位职责。
- 要素分析法：拆分核心、辅助、约束要素，结构简单，适配各类基础业务分析。
# 五、安全风险类
- 风险矩阵：结合发生概率与危害程度，划分风险等级，实行分级管控。
- LEC 评价法：量化作业危险程度，实现安全风险打分、定量评价。
- JSA 作业安全分析：拆解作业步骤，分步识别风险，规范现场安全操作。
# 六、综合通用类
- 辩证分析法：一分为二看待问题，客观阐述利弊、正反两面，适用性广。
- 对标分析法：对照标准、标杆找差距，查漏补缺，推动整改提升。
- 趋势分析法：依托历史数据，研判发展规律与未来变化走向。
- 成本效益分析：对比投入与产出，科学比对方案、辅助决策。
